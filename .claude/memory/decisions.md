# Решения проекта

Записывай решения, которые потом будет неочевидно восстановить из кода.
Формат: **Что:** + **Why:** + **Дата.**

## Шаблон записи

```markdown
### [YYYY-MM-DD] [Краткая суть решения]

**Что:** [решение в 1-2 предложениях]

**Why:** [причина — ограничение, требование заказчика, технический trade-off,
прошлый инцидент. Без этого через месяц не понять зачем сделано так.]

**Альтернатива:** [что рассматривали и почему отбросили — опционально]
```
